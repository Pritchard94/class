20 HTML ELEMENTS (TAGS)

  1. TABLE - This Element is used to define a table. It is used as <table ></table >

  2. TR- The TR element is used to define a row in a table. It is used as <tr></tr>

  3. TH- The TH element is used to define a header in a table. It is used as <th></th>

  4. TD- The TD element is used to define a cell in a table. It is used as <td></td>

  5. TEMPLATE- It is used to hide certain HTML content from the user when the page loads.It is used as <template></template>

  6. PROGRESS- It is used to represent the completion progress of a task. It is used as <progress></progrss>

  7. METER- It is used to display a scalar value within a given range. It is used as <meter></meter>

  8. DIALOG- It is used to open up a dialog box. It is used as <dialog></dialog>

  9. ADDRESS- It is used to define the address of the owner of an article. It is used as <address></address>

  10. AUDIO- It is used to embed sound content in the document. It is used as <audio></audio>

  11. BASE- It specifies base URL. It has no closing tag. It is used as <base>

  12. CAPTION- It specifies a table's caption. It is used as <caption></caption>

  13. CODE- It is used to define a computer code. It is used as <code></code>

  14. DETAILS- Specifies additional details tha the user can open or close on demand. It is used as <details></details>

  15. MAP- It is used to define an image map. It is used as <map></map>

  16. OBJECT- defines a container for external resources. It is used as <object></object>

  17. PARAM- used to define parameter for an object element. It has no closing tag

  18. SUMMARY- defines a visible header for a detail element. It is used as <summary></summary>

  19. SVG- defines a container for a svg graphics. It is used as <svg></svg>

  20. TIME- defines a specific time. It is used as <time></time>
